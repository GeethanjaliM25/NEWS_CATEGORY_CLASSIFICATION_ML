# 📰 News Article Categorization using Naive Bayes

This is a full-stack machine learning project that classifies news articles into different categories using the **Naive Bayes algorithm**

### 🎯 Categories

* **1 – World**
* **2 – Sports**
* **3 – Business**
* **4 – Sci/Tech**

---

## 🚀 Project Features

* Uses **Multinomial Naive Bayes** for text classification
* Frontend built using **HTML, CSS, JavaScript**
* Backend built using **Flask (Python)**
* Dataset taken from **Kaggle**
* Real-time prediction from user input
* Model accuracy: **~90%**

---

## ▶️ How to Run

### Start the backend

```bash
python app.py
```

### Start the frontend

```bash
python -m http.server 5500
```

Open in browser:

```
http://localhost:5500
```

---

## 🧪 Example

**Input:**

```
India won 5 medals in Olympics
```

**Output:**

```
Predicted Category: Sports
Class ID: 2
```
